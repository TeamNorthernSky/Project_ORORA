# 프로젝트 규칙

## 작업 규칙

- 사용자가 "구현해 주세요", "작성해 주세요", "수정해 주세요" 등 명시적으로 요청하기 전까지는 코드와 프로젝트 파일을 절대 수정하지 않는다.
- 명시적 요청이 없는 경우, 분석·확인·제안만 진행한다.

## UI 구현 규칙

- 배경(Image)과 텍스트(TextMeshProUGUI)가 함께 필요한 UI 요소는, **부모에 Image, 자식에 Text** 구조로 생성한다.
- 예시: `GoldLabel`(Image, 배경) > `GoldLabelText`(TextMeshProUGUI)
- `~Group` 빈 오브젝트로 묶는 방식은, 위 방식이 불가능한 특수한 경우에만 사용한다.
